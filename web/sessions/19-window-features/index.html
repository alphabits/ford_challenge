Session 19: Window Features¶

Start time:
18-05-2011 22:09
End time:
18-05-2011 23:40

In this session I try to test whether a better performance can be achieved, when the running features are exchanged with windows instead. It is more or less a copy of the work done in Session 17: Recreating Winning Entry and Session 18: Forward Selection.
